# Nv-pro

下载下来后，安装相关依赖

> clone相关代码

> $ yarn install

> $ gulp

### 主题更换和支持

默认是`极客蓝`，另外还支持`经典红`、`幽暗黑`、`深空灰`、`活力橙`

更换很简单

**极客蓝**

```html
    <!--css样式请放在head标签内部-->
    <link type="text/css" rel="stylesheet" data-theme href="//storage.360buyimg.com/v2.0.0/nv.min.css" />
```

***切记，`data-theme`必须添加，否则组件库无法识别当前的主题色***
